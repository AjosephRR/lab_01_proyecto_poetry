# Lab 03 - Pruebas y TDD

## Historia
Como sistema de órdenes, quiero calcular el total final de una compra, validar descuentos y notificar la compra procesada, para asegurar que la lógica sea correcta y confiable.

## Qué incluye
- pytest
- fixtures
- parametrización
- markers
- mocking con unittest.mock
- property-based testing con Hypothesis
- cobertura con pytest-cov
