# Lab 02 - Fundamentos del lenguaje

## Objetivo
Este laboratorio practica los fundamentos del lenguaje en Python mediante un script que:

- lee un archivo JSON,
- filtra usuarios activos,
- agrupa usuarios por ciudad,
- valida correos con expresiones regulares,
- clasifica edades con pattern matching,
- y maneja errores de archivo o formato.

## Temas aplicados
- Sintaxis e indentación
- Variables y tipos de datos
- Colecciones (`list`, `dict`)
- Control de flujo (`if`, `for`, `match/case`)
- Manejo de excepciones (`try-except`)
- Expresiones regulares
- Lectura de archivos JSON

## Estructura
- `data/usuarios.json`: datos de entrada
- `src/main.py`: script principal del laboratorio

## Cómo ejecutarlo
Ubicarse en la carpeta del laboratorio y correr:

```powershell
python .\src\main.py
