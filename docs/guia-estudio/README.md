# Guia de Estudio de los Modulos Principales

Esta carpeta contiene una guia por cada modulo principal del curso:

- `guia-fundamental.md`
- `guia-intermedio.md`
- `guia-avanzado.md`

La idea es que puedas usar estos documentos para:

- estudiar antes de una exposicion
- entender como evoluciona el curso
- explicar cada modulo con ejemplos reales del proyecto actual
- repasar conceptos y terminologia sin lenguaje tecnico innecesario

Orden sugerido de lectura:

1. `guia-fundamental.md`
2. `guia-intermedio.md`
3. `guia-avanzado.md`

Frase corta para presentar el curso completo:

`fundamental` ensena las bases del lenguaje y las herramientas, `intermedio` ensena a construir aplicaciones reales con mejor estructura, y `avanzado` ensena a trabajar con arquitectura, automatizacion, seguridad y mantenimiento.
